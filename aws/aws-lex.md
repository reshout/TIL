# Amazon Lex

- 음성 및 텍스트를 사용한 대화형 인터페이스를 애플리케이션에 쉽게 추가할 수 있도록 도와주는 서비스
- 자동 음성 인식(ASR) + 자연어 이해(NLU) + 딥러닝
- Alexa와 동일한 딥러닝 기술 적용
- AWS Lambda, MobileHub, CloudWatch와 통합
